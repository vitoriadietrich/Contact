# Responsive Contact Us Form in HTML And CSS

Um projeto simples de **formulário de contato responsivo** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **formulário de contato moderno**, adaptado para diferentes tamanhos de tela.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Layout responsivo
- Campos de contato (nome, email, mensagem)
- Design simples e limpo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
